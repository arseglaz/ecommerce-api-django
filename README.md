# E-commerce API

REST API for an online store based on Django and Django REST Framework.

## Technologies

- **Backend**: Django 5.0.2, Django REST Framework 3.14
- **Authentication**: JWT (djangorestframework-simplejwt)
- **Database**: PostgreSQL (dev: SQLite)
- **Python**: 3.10+
